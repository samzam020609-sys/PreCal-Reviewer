01 / Equation of a Line via y = mx + b

Slope-Intercept Form, Slope Form, General Form — and how to substitute into each

📐 The Three Forms at a Glance
NameEquationWhen to use
Slope-Intercept (SIF)y = mx + bSlope and y-intercept are known; easiest to graph
Slope Form (SF) / Point-Slopey − y₁ = m(x − x₁)Slope + one point known; intermediate step
General Form (GF)Ax + By + C = 0Final clean answer; A, B, C are integers, A ≥ 0
Two-Point Form(y−y₁)/(y₂−y₁) = (x−x₁)/(x₂−x₁)Two points given; derive everything from them
Intercept Formx/a + y/b = 1Both intercepts are known
🔑 Key Variables Decoded
  • m (slope)Rise ÷ Run = Δy / Δx. Measures how steep the line is and in which direction it tilts.
  • b (y-intercept)The y-value when x = 0. It's where the line "starts" on the vertical axis.
  • x₁, y₁A specific known point the line passes through.
  • A, B, CIntegers in General Form. A must be non-negative; no fractions or decimals allowed.
To go from SIF → GF: move all terms to the left side and clear fractions by multiplying through. Example: y = (2/3)x − 4 → multiply by 3 → 3y = 2x − 12 → 2x − 3y − 12 = 0
⚙️ Substituting Into y = mx + b (Step-by-Step)

Use this when you know the slope m and one point (x₁, y₁):

  1. Write the base equation: y = mx + b
  2. Substitute the known m, x₁, and y₁ into the equation
  3. Solve for b (y-intercept)
  4. Rewrite the final equation with m and b
  5. Convert to General Form if required: rearrange so everything is on one side = 0
Given: m = 3, passes through (2, 7)
y = 3x + b → 7 = 3(2) + b → 7 = 6 + b → b = 1
SIF: y = 3x + 1  |  GF: 3x − y + 1 = 0
Given: m = −2, passes through (−1, 4)
y = −2x + b → 4 = −2(−1) + b → 4 = 2 + b → b = 2
SIF: y = −2x + 2  |  GF: 2x + y − 2 = 0
Given: m = 2/3, passes through (3, 5)
y = (2/3)x + b → 5 = (2/3)(3) + b → 5 = 2 + b → b = 3
SIF: y = (2/3)x + 3 → multiply by 3 → GF: 2x − 3y + 9 = 0
Fraction slope? Multiply the entire equation by the denominator to get rid of fractions before writing GF.

02 / Equation Given Two Points

Finding the line equation when (x₁, y₁) and (x₂, y₂) are given

📍 The Core Formula

First compute the slope, then find b using either point.

m = (y₂ − y₁) / (x₂ − x₁)
then use: y − y₁ = m(x − x₁)  or  substitute into y = mx + b
Which point do you use to find b? Either one gives the same answer — pick whichever has nicer numbers (smaller values, no negatives if possible).
📋 Full Method (Step-by-Step)
  1. Label your points clearly: (x₁, y₁) and (x₂, y₂)
  2. Compute slope: m = (y₂ − y₁) / (x₂ − x₁)
  3. Substitute m and one point into y = mx + b, then solve for b
  4. Write the equation in SIF: y = mx + b
  5. If needed, convert to General Form: Ax + By + C = 0
  6. Verify by substituting both original points — both must satisfy the equation
Don't mix up your x and y values! Always subtract in the same order: (y₂ − y₁) and (x₂ − x₁). Swapping order on both top and bottom is fine — the result is the same. But never subtract in different orders: (y₂ − y₁)/(x₁ − x₂) is WRONG.
✏️ Worked Examples
Points: (1, 3) and (4, 9)
m = (9−3)/(4−1) = 6/3 = 2
Use (1,3): 3 = 2(1) + b → b = 1
y = 2x + 1  |  GF: 2x − y + 1 = 0
Verify: (4,9) → 9 = 2(4)+1 = 9 ✓
Points: (−2, 5) and (3, −5)
m = (−5−5)/(3−(−2)) = −10/5 = −2
Use (3,−5): −5 = −2(3) + b → −5 = −6 + b → b = 1
y = −2x + 1  |  GF: 2x + y − 1 = 0
Verify: (−2,5) → 5 = −2(−2)+1 = 5 ✓
Points: (0, 4) and (6, 7)
m = (7−4)/(6−0) = 3/6 = 1/2
Since x₁ = 0, that means (0, 4) is the y-intercept: b = 4 directly!
y = (1/2)x + 4 → multiply by 2 → GF: x − 2y + 8 = 0
If one point is (0, k), then b = k immediately. No calculation needed!
Points: (−3, 5) and (7, 5)
m = (5−5)/(7−(−3)) = 0/10 = 0
Slope is zero → horizontal line! Just use y = the shared y-value.
y = 5  |  GF: y − 5 = 0
⚡ Two-Point Form (Skip the Slope Step)

An alternate formula that works directly from two points:

(y − y₁) / (y₂ − y₁) = (x − x₁) / (x₂ − x₁)

Cross-multiply and simplify to get the equation directly.

Points: (2, 1) and (5, 7)
(y − 1)/(7 − 1) = (x − 2)/(5 − 2)
(y − 1)/6 = (x − 2)/3
Cross-multiply: 3(y − 1) = 6(x − 2)
3y − 3 = 6x − 12
y = 2x − 3  |  GF: 2x − y − 3 = 0
Two-Point Form is fastest in exams. Memorize it: set up the ratio of differences, cross-multiply, simplify.

03 / Equation Given x & y Intercepts

When the x-intercept (a) and y-intercept (b) are both given

🎯 What Intercepts Give You
  • x-intercept = aThe point (a, 0) — y is always 0 on the x-axis
  • y-intercept = bThe point (0, b) — x is always 0 on the y-axis. This directly gives you b in y = mx + b!
The y-intercept IS b. So if the y-intercept = 6, you already know b = 6. You only need to compute the slope.
📐 Intercept Form (Fastest Method)

When both intercepts are non-zero, plug directly into the Intercept Form:

x / a + y / b = 1

Where a = x-intercept and b = y-intercept.

To convert to Standard/GF: multiply everything by ab (the LCD), then rearrange to Ax + By = C or Ax + By + C = 0.
This formula breaks if either intercept is 0! A zero x-intercept means the line passes through the origin — use a different method.
🔄 Method via Slope + Substitution
  1. The two known points are (a, 0) and (0, b)
  2. Compute slope: m = (b − 0) / (0 − a) = −b/a
  3. Since (0, b) is the y-intercept, the y-intercept is already b. Write: y = (−b/a)x + b
  4. Simplify or convert to General Form as needed
✏️ Worked Examples
x-intercept = 4, y-intercept = 6
Points: (4, 0) and (0, 6)
Via Intercept Form:
x/4 + y/6 = 1 → multiply by 12 → 3x + 2y = 12
GF: 3x + 2y − 12 = 0  |  SIF: y = −(3/2)x + 6
Verify: (4,0) → 3(4)+2(0)=12 ✓   (0,6) → 3(0)+2(6)=12 ✓
x-intercept = −3, y-intercept = 5
Points: (−3, 0) and (0, 5)
Slope: m = (5−0)/(0−(−3)) = 5/3
y = (5/3)x + 5 → multiply by 3 → 3y = 5x + 15
GF: 5x − 3y + 15 = 0
Quick intercept form check: x/−3 + y/5 = 1 → multiply by −15 → 5x − 3y = −15 → 5x − 3y + 15 = 0 ✓
x-intercept = 2, y-intercept = 3
x/2 + y/3 = 1 → multiply by 6 → 3x + 2y = 6
GF: 3x + 2y − 6 = 0  |  SIF: y = −(3/2)x + 3
📋 Intercept Form vs Two-Point Method
MethodBest WhenWatch Out For
Intercept Form: x/a + y/b = 1Both intercepts are clean integersFails if either = 0
Two-Point via slope + substitutionAlways worksMore steps, easy sign errors
Golden shortcut for GF from intercepts: If x-int = a and y-int = b, then GF is always: bx + ay = ab.
Example: a=4, b=6 → 6x + 4y = 24 → divide by 2 → 3x + 2y = 12. Done!
04 / Graphing System of Linear Equations

Plotting two lines and reading where they meet

📊 What Is a System?

A system of linear equations is two or more equations considered at the same time. The solution is the ordered pair (x, y) that satisfies ALL equations simultaneously — it's the point where the graphs intersect.

  • One solutionLines intersect at exactly one point → consistent & independent
  • No solutionLines are parallel → inconsistent
  • Infinite solutionsLines are the same (overlapping) → consistent & dependent
📝 Steps to Graph a System
  1. Convert each equation to Slope-Intercept Form: y = mx + b
  2. For each line: plot the y-intercept (0, b) on the y-axis
  3. From the y-intercept, apply the slope (rise/run) to find a second point
  4. Draw a straight line through those two points — extend it with arrows
  5. Repeat for the second equation on the same coordinate plane
  6. Identify the intersection point — that is your solution (x, y)
  7. Verify by substituting the intersection into both original equations
3-point rule: Always plot at least 3 points per line when graphing by hand. One point to start (y-intercept), one from slope, one more as a check. A single mistake in slope direction is hard to catch with only 2 points.
📈 Graph Example

System: y = x + 1  and  y = −x + 5

Intersection → solution (2, 3)
Verify: (2,3) in y=x+1 → 3=2+1 ✓  |  (2,3) in y=−x+5 → 3=−2+5 ✓
📌 How to Plot Using Slope (Visual Method)
Positive slope (m = 2/3)
Start at y-intercept → go UP 2, RIGHT 3 → mark point → draw line
Negative slope (m = −3/4)
Start at y-intercept → go DOWN 3, RIGHT 4 → mark point → draw line
Intercept method for graphing: Instead of using slope, plot both the x-intercept (let y=0, solve for x) and y-intercept (let x=0, solve for y). Connect the two intercepts — done in 2 steps!
📋 Reading the Graph — What to Expect
What you see on graphConclusionType of system
Two lines crossing at one pointOne solution (x, y)Consistent & Independent
Two parallel lines, never touchNo solutionInconsistent
One line (both equations plot same line)Infinitely many solutionsConsistent & Dependent
Graphing is the least precise method — intersection points with non-integer coordinates are hard to read accurately. When exactness is needed, use Elimination or Substitution.
05 / Solving by Elimination

Cancel one variable by combining equations — then solve for the other

💡 The Core Idea

The Elimination Method (also called Addition Method) works by making the coefficient of one variable opposites in the two equations. When you add the equations, that variable disappears.

If: 3x + 5y = 20  and  3x − 5y = 4
→ Add: 6x = 24 → x = 4  (y eliminated!)
Rule of thumb: Eliminate whichever variable is easiest — usually the one whose coefficients are already equal or easy to make equal with a single multiplication.
📋 Full Step-by-Step Process
  1. Write both equations in standard form: Ax + By = C (no variables on the right side)
  2. Choose which variable to eliminate (x or y)
  3. If needed, multiply one or both equations by a constant to make those coefficients opposite numbers (e.g., +4 and −4)
  4. Add the two equations. One variable cancels.
  5. Solve the resulting one-variable equation
  6. Substitute the solved value back into either original equation to find the other variable
  7. Write the answer as an ordered pair (x, y)
  8. Check by substituting into both original equations
✏️ Worked Examples
System: 2x + 3y = 12 and 4x − 3y = 6
y-coefficients: +3 and −3 → already opposites!
Add: (2x+4x) + (3y−3y) = 12+6 → 6x = 18 → x = 3
Sub x=3 into eq.1: 2(3)+3y=12 → 6+3y=12 → 3y=6 → y=2
Solution: (3, 2)
Check: eq.2 → 4(3)−3(2)=12−6=6 ✓
System: 3x + 2y = 11 and x + y = 4
Eliminate x: multiply eq.2 by −3 → −3x − 3y = −12
Add: (3x−3x)+(2y−3y)=11−12 → −y = −1 → y = 1
Sub y=1 into eq.2: x+1=4 → x=3
Solution: (3, 1)
Check eq.1: 3(3)+2(1)=9+2=11 ✓
System: 2x + 3y = 7 and 3x + 5y = 11
Eliminate x: multiply eq.1 by 3, eq.2 by −2
→ 6x + 9y = 21 and −6x − 10y = −22
Add: −y = −1 → y = 1
Sub y=1 into eq.1: 2x+3(1)=7 → 2x=4 → x=2
Solution: (2, 1)
To eliminate x: find LCM of x-coefficients (2 and 3 → LCM=6). Multiply eq.1 by 3, eq.2 by −2.
System: 2x + 4y = 8 and x + 2y = 7
Multiply eq.2 by −2: → −2x − 4y = −14
Add: 0x + 0y = 8 + (−14) → 0 = −6
This is false → the system has no solution (parallel lines)!
System: 2x + 4y = 8 and x + 2y = 4
Multiply eq.2 by −2: → −2x − 4y = −8
Add: 0x + 0y = 8 + (−8) → 0 = 0

🧮 Choosing What to Eliminate
SituationStrategy
Coefficients already opposite: +3 and −3Just add directly — no multiplication needed
Same coefficient: +4 and +4Multiply one eq. by −1, then add
One is a multiple: 2 and 6Multiply the eq. with 2 by −3, add
No obvious relationship: 3 and 5Use LCM: multiply eq.1 by 5, eq.2 by −3 (or vice versa)
LCM Shortcut: To eliminate variable with coefficients a and b, multiply eq.1 by b and eq.2 by −a (or vice versa). Example: coefficients 4 and 6 → multiply by 6 and −4, giving 24 and −24.
🏁 Interpreting Results at End of Elimination

x = number (or y = number)One solution existsBack-substitute to find the other variable

After finding one variable, always substitute back into an original equation, not the manipulated one, to avoid errors.

The fastest check: Convert both equations to SIF and just compare slopes (m) and y-intercepts (b). Same m, different b → parallel. Same m, same b → overlapping. Different m → intersecting.
∥ Parallel Lines — Deep Dive

Parallel lines have the same slope but different y-intercepts. They run in the same direction, maintaining a constant distance between them — never meeting.

m₁ = m₂  and  b₁ ≠ b₂
Are y = 3x + 2 and y = 3x − 7 parallel?
Both have m = 3. y-intercepts: 2 ≠ −7. → Yes, parallel! No solution.
Are 2x − 4y + 8 = 0 and x − 2y + 1 = 0 parallel?
Convert to SIF:
Eq.1: −4y = −2x − 8 → y = (1/2)x + 2
Eq.2: −2y = −x − 1 → y = (1/2)x + (1/2)
Same slope (1/2), different b → Parallel! Inconsistent.
A line parallel to y = mx + b through point (x₁, y₁) has the same slope m. Just substitute to find b: y₁ = m·x₁ + b → solve for b.
⊥ Perpendicular Lines — Deep Dive

Two lines are perpendicular if their slopes are negative reciprocals. They meet at exactly one point and form a 90° angle.

m₁ × m₂ = −1   →   m₂ = −1/m₁

To get the negative reciprocal: flip the fraction and change the sign.

Slope m₁Perpendicular slope m₂Check: m₁×m₂
2−1/22 × (−1/2) = −1 ✓
3/4−4/3(3/4)(−4/3) = −1 ✓
−51/5(−5)(1/5) = −1 ✓
1−11×(−1) = −1 ✓
−2/33/2(−2/3)(3/2) = −1 ✓
Finding a perpendicular line through a point:
1) Take the negative reciprocal of the given slope.
2) Use y − y₁ = m_new(x − x₁) with the new slope and given point.
Find the line perpendicular to y = 2x + 5, passing through (4, 1)
Slope of given line: m = 2
Perpendicular slope: m⊥ = −1/2
Use point-slope: y − 1 = −(1/2)(x − 4)
y − 1 = −(1/2)x + 2
y = −(1/2)x + 3  |  GF: x + 2y − 6 = 0
↔️ Overlapping (Coincident) Lines

When two different-looking equations actually represent the same line. Every point on the line is a solution.

This happens when one equation is just a scalar multiple of the other.

y = 2x + 4 and 4x − 2y + 8 = 0
Convert eq.2: −2y = −4x − 8 → y = 2x + 4
Identical! → Infinitely many solutions (dependent system)
3x + 6y = 12 and x + 2y = 4
Divide eq.1 by 3: x + 2y = 4 → identical to eq.2!
Same line → Infinite solutions
Quick check: divide one equation by the other's coefficients. If all ratios are equal (A₁/A₂ = B₁/B₂ = C₁/C₂), the lines are the same.
👁️ Visual Summary of All Four Types
⚡ All Shortcuts in One Place
Check if parallel: Compare slopes. If equal and b's differ → parallel.
Check if perpendicular: Multiply slopes. If product = −1 → perpendicular.
Check if overlapping: Write in SIF. If both equations are literally identical → overlapping.
Find perpendicular slope: Flip the fraction and negate. E.g., 3/5 → −5/3.
A vertical line (x = a) is perpendicular to any horizontal line (y = b). Vertical lines have undefined slope, so the m₁×m₂=−1 rule doesn't apply — but they ARE perpendicular geometrically.
07 / Circles

Distance Formula · Midpoint Formula · Standard & General Form · Completing the Square

📏 Distance Formula

The distance between two points is derived from the Pythagorean Theorem. The horizontal distance (Δx) and vertical distance (Δy) form the legs of a right triangle; the distance is the hypotenuse.

d = √[(x₂ − x₁)² + (y₂ − y₁)²]
Order doesn't matter: (x₂−x₁)² = (x₁−x₂)² because squaring removes the negative. Always subtract in the same order (2 minus 1), but either order gives the same answer.
Points: (1, 2) and (4, 6)
d = √[(4−1)² + (6−2)²] = √[9 + 16] = √25 = 5
Points: (−3, 1) and (5, −5)
d = √[(5−(−3))² + (−5−1)²] = √[64 + 36] = √100 = 10
Points: (2, 3) and (7, 15) — leave in radical form
d = √[(7−2)² + (15−3)²] = √[25 + 144] = √169 = 13
➕ Midpoint Formula

The midpoint is simply the average of the x-coordinates and the average of the y-coordinates.

M = ((x₁ + x₂)/2 , (y₁ + y₂)/2)
Memory trick: "Middle = Average." Add both x's and divide by 2. Add both y's and divide by 2.
Midpoint of (2, 4) and (8, 10)
M = ((2+8)/2, (4+10)/2) = (5, 7)
Midpoint of (−3, 5) and (7, −1)
M = ((−3+7)/2, (5+(−1))/2) = (4/2, 4/2) = (2, 2)
Midpoint is (3, 5). One endpoint is (1, 2). Find the other.
Let other endpoint = (x₂, y₂)
(1+x₂)/2 = 3 → x₂ = 5
(2+y₂)/2 = 5 → y₂ = 8
Other endpoint: (5, 8)
Find missing endpoint: double the midpoint coords, then subtract the known endpoint. x₂ = 2(3)−1 = 5, y₂ = 2(5)−2 = 8.
⭕ Standard Form of a Circle

A circle with center (h, k) and radius r:

(x − h)² + (y − k)² = r²
  • (h, k)Center — read directly from the equation (flip signs!)
  • rRadius — take the square root of the right-hand side
Sign trap! (x − 3)² means h = +3, NOT −3. (y + 2)² = (y − (−2))² means k = −2, NOT +2. Always flip the sign when reading center from standard form.
Center (3, −2), radius = 5
(x − 3)² + (y − (−2))² = 5² → (x − 3)² + (y + 2)² = 25
Read from: (x + 4)² + (y − 1)² = 49
h = −4, k = 1 (flip the signs!) → Center = (−4, 1)
r² = 49 → r = 7
Center: (−4, 1)  |  Radius: 7
Circle centered at origin, radius = √10
h=0, k=0 → special case:
x² + y² = 10
🔁 General Form → Standard Form (Completing the Square)
x² + y² + Dx + Ey + F = 0
  1. Group x-terms and y-terms: (x² + Dx) + (y² + Ey) = −F
  2. Complete the square for x: add (D/2)² to both sides
  3. Complete the square for y: add (E/2)² to both sides
  4. Factor the left side into perfect squares
  5. Identify center (h, k) and radius r
Completing the square trick: Take the coefficient of x (or y), halve it, square it — add to both sides.
x² + 6x → take 6, halve → 3, square → 9 → add 9: (x² + 6x + 9) = (x + 3)²
x² + y² − 6x + 4y − 12 = 0
Group: (x² − 6x) + (y² + 4y) = 12
Complete x: add (3)² = 9 → (x−3)²
Complete y: add (2)² = 4 → (y+2)²
(x−3)² + (y+2)² = 12 + 9 + 4 = 25
Center: (3, −2)  |  r = 5
x² + y² + 10x − 2y + 1 = 0
Group: (x²+10x) + (y²−2y) = −1
Complete x: (10/2)²=25 → add 25: (x+5)²
Complete y: (2/2)²=1 → add 1: (y−1)²
(x+5)² + (y−1)² = −1 + 25 + 1 = 25
Center: (−5, 1)  |  r = 5
2x² + 2y² − 8x + 12y − 24 = 0
Divide everything by 2 first: x² + y² − 4x + 6y − 12 = 0
Group: (x²−4x) + (y²+6y) = 12
Complete x: add 4 → (x−2)²
Complete y: add 9 → (y+3)²
(x−2)² + (y+3)² = 12+4+9 = 25
Center: (2, −3)  |  r = 5
When leading coefficient ≠ 1, always divide by it first before completing the square!
🗺️ Given Diameter Endpoints → Circle Equation
  1. Use Midpoint Formula to find the center (midpoint of diameter)
  2. Use Distance Formula to find diameter length, then r = diameter/2
    OR find distance from center to either endpoint — that's the radius directly
  3. Substitute (h, k) and r into Standard Form
Diameter endpoints: (1, 2) and (5, 8)
Center = midpoint: ((1+5)/2, (2+8)/2) = (3, 5)
r = distance from (3,5) to (5,8):
r = √[(5−3)² + (8−5)²] = √[4+9] = √13
(x−3)² + (y−5)² = 13
Diameter endpoints: (−4, 0) and (4, 6)
Center = ((−4+4)/2, (0+6)/2) = (0, 3)
r = distance from (0,3) to (4,6):
r = √[16+9] = √25 = 5
x² + (y−3)² = 25

🪄 What Is a Parabola?

A parabola is the set of all points in a plane that are equidistant from a fixed point called the focus and a fixed line called the directrix. That equal distance is called p (or sometimes a or c).

distance from point to Focus = distance from point to Directrix
  • Vertex (V)The turning point of the parabola — the point exactly halfway between the focus and directrix. It's the "tip" of the curve.
  • Focus (F)A fixed point inside the curve, at distance p from the vertex. The parabola wraps around it.
  • DirectrixA fixed line outside the curve, also at distance p from the vertex. Perpendicular to the axis of symmetry.
  • p (focal distance)The distance from the vertex to the focus (and from the vertex to the directrix). Also written as a in some books.
  • Axis of Symmetry (AOS)The line passing through the vertex and focus. Divides the parabola into two mirror-image halves.
  • Latus Rectum (LR)The chord through the focus, perpendicular to the axis of symmetry. Its length is always |4p|.
  • Endpoints of LRThe two points where the latus rectum intersects the parabola. Each is at distance |2p| from the focus, on either side.
  • Principal AxisSame as the Axis of Symmetry — the line the parabola is symmetric about. For vertical parabolas it's a vertical line; for horizontal it's a horizontal line.
🧭 The Four Orientations — Vertex at Origin

When the vertex is at the origin (0, 0), the four standard equations are:

OpeningEquationFocusDirectrixAOSLR length
⬆ Upward x² = 4py (0, p) y = −p x = 0 (y-axis) |4p|
⬇ Downward x² = −4py (0, −p) y = p x = 0 (y-axis) |4p|
➡ Right y² = 4px (p, 0) x = −p y = 0 (x-axis) |4p|
⬅ Left y² = −4px (−p, 0) x = p y = 0 (x-axis) |4p|
Which variable is squared?
x² = parabola opens up or down (vertical axis)
y² = parabola opens left or right (horizontal axis)

Which way does it open?
Positive coefficient of 4p → opens UP (x²) or RIGHT (y²)
Negative coefficient → opens DOWN (x²) or LEFT (y²)
📍 Vertex NOT at Origin — General Standard Form

When the vertex is at (h, k), just shift everything by (h, k):

OpeningEquationFocusDirectrixAOS
⬆ Upward (x−h)² = 4p(y−k) (h, k+p) y = k−p x = h
⬇ Downward (x−h)² = −4p(y−k) (h, k−p) y = k+p x = h
➡ Right (y−k)² = 4p(x−h) (h+p, k) x = h−p y = k
⬅ Left (y−k)² = −4p(x−h) (h−p, k) x = h+p y = k
Pattern for Focus: The focus is always p units away from the vertex in the opening direction.
Opens up → add p to k. Opens right → add p to h. Opens down → subtract p from k. Opens left → subtract p from h.
p is always positive! The sign in the equation (±4p) tells you the direction, but p itself is a positive number. Always extract p as a positive value, then apply direction separately.
🔑 How to Extract Every Part From an Equation
  1. Identify which variable is squared. Squared x → vertical parabola (up/down). Squared y → horizontal (left/right).
  2. Read the vertex (h, k) from the equation. Remember: (x−h)² and (y−k)² — flip the signs.
  3. Find 4p. The coefficient on the right side (the non-squared side) equals 4p. Solve: p = that number ÷ 4. If negative, note the direction but p = |value ÷ 4|.
  4. Determine opening direction from the sign: positive → up or right; negative → down or left.
  5. Find the focus by moving p units from vertex in the opening direction.
  6. Find the directrix by moving p units from vertex in the opposite direction.
  7. Write the AOS (Principal Axis). Vertical parabola: x = h. Horizontal: y = k.
  8. Find endpoints of latus rectum. The LR passes through the focus, perpendicular to the AOS, with half-length |2p| on each side. So endpoints are ±2p from the focus, perpendicular to the axis.

🔁 General Form → Standard Form (Complete the Square)

Parabolas in General Form look like Ax² + Dx + Ey + F = 0 (vertical) or Cy² + Dx + Ey + F = 0 (horizontal). Convert by completing the square on the squared term.

  1. Isolate the squared-variable terms on one side and the linear terms on the other
  2. Complete the square on the squared variable (group it, add (half the coeff)² to both sides)
  3. Factor into a perfect square: (x−h)² or (y−k)²
  4. Divide/simplify until it matches (x−h)²=4p(y−k) or (y−k)²=4p(x−h)
Convert: x² − 4x − 8y + 20 = 0
Move non-x² terms: x² − 4x = 8y − 20
Complete the square: (x² − 4x + 4) = 8y − 20 + 4
(x − 2)² = 8y − 16 = 8(y − 2)
Standard form: (x−2)² = 8(y−2)
→ h=2, k=2, 4p=8, p=2, opens UP
Vertex: (2,2) | Focus: (2,4) | Directrix: y=0 | AOS: x=2 | LR: 8 units

🥚 What Is an Ellipse?

An ellipse is the set of all points in a plane such that the sum of the distances from two fixed points (the foci) is constant. Think of it as a "stretched circle" — when both foci are at the same point, you get a circle.

d(P, F₁) + d(P, F₂) = 2a  ←  constant sum, always equals 2a

Imagine pinning two thumbtacks (the foci) on paper, looping a string around them, and tracing with a pencil held taut — the path traced is an ellipse.

The defining property: d₁ + d₂ = 2a for every point on the curve
Real-world examples: planetary orbits, whispering galleries, lithotripsy machines, satellite dishes. The planets orbit the Sun in elliptical paths with the Sun at one focus (Kepler's First Law).
🔑 Every Part of an Ellipse — Defined
  • Center (h, k)The exact middle of the ellipse — midpoint of both axes. Origin (0,0) if not shifted.
  • a (semi-major)Half the length of the major axis. Always the largest value under a fraction. Distance from center to each vertex.
  • b (semi-minor)Half the length of the minor axis. Always the smaller value. Distance from center to each co-vertex.
  • c (focal distance)Distance from the center to each focus. Found using: c² = a² − b²
  • VerticesThe two endpoints of the major axis — the "tips" of the longer direction. Each is a units from center.
  • Co-verticesThe two endpoints of the minor axis. Each is b units from center, perpendicular to vertices.
  • Foci (F₁, F₂)Two fixed interior points on the major axis, each c units from the center. The defining points of the ellipse.
  • Major AxisThe longest diameter — passes through both foci and both vertices. Length = 2a.
  • Minor AxisThe shorter diameter — perpendicular to major axis at the center. Length = 2b.
  • Latus RectumA chord through each focus, perpendicular to the major axis. Length of each = 2b²/a.
All parts of an ellipse at a glance — horizontal orientation
● Vertices — on major axis, ±a from center ● Co-vertices — on minor axis, ±b from center ● Foci — on major axis, ±c from center ● Latus Rectum — through focus, length 2b²/a
The golden rule: a > b always. The bigger denominator under the fractions tells you which direction the major axis goes. Bigger under x² → horizontal major axis. Bigger under y² → vertical major axis.
⚡ The Master Relationship: a² = b² + c²

All three measurements are connected through one Pythagorean-style relationship:

a² = b² + c²   →   c² = a² − b²   →   b² = a² − c²
Given a and b → find c
c = √(a² − b²)
Most common in problems.
Given a and c → find b
b = √(a² − c²)
Used when foci and vertices are given.
a, b, c form a right triangle — hypotenuse is always a
a is ALWAYS the largest. If you ever get c > a or b > a, you made an error. The foci are always inside the ellipse, so c < a.

🔄 How to Extract All Parts From an Equation
  1. Make sure the equation = 1. If it's not, divide everything by the constant on the right.
  2. Identify the center (h, k) from the equation. Remember to flip signs: (x−3)² → h=3, (y+2)² → k=−2.
  3. Read a² and b² from the denominators. a² is the larger denominator — take √ to get a and b.
  4. Find c using c² = a² − b², then c = √(a²−b²).
  5. Determine orientation: bigger denominator under x² → horizontal; under y² → vertical.
  6. Place vertices at ±a from center along the major axis direction.
  7. Place co-vertices at ±b from center along the minor axis direction.
  8. Place foci at ±c from center along the major axis direction.
  9. Compute latus rectum length = 2b²/a (optional, if asked).
The 3-step speed method:
1) Identify a², b² (bigger = a²) and direction
2) Find c = √(a²−b²)
3) All points = center ± the right value in the right direction
Everything follows from knowing a, b, c, and the center.

🔁 General Form → Standard Form (Complete the Square)

Ellipses in general form look like Ax² + By² + Cx + Dy + E = 0 where A ≠ B but both are positive. Convert by completing the square on both variables.

  1. Group x-terms and y-terms: (Ax²+Cx) + (By²+Dy) = −E
  2. Factor out the leading coefficient from each group
  3. Complete the square inside each group (add inside, compensate outside)
  4. Write as perfect squares: A(x−h)² + B(y−k)²= some constant
  5. Divide both sides by that constant to get = 1
Convert: 4x² + 9y² − 24x + 36y + 36 = 0
Group: (4x²−24x) + (9y²+36y) = −36
Factor: 4(x²−6x) + 9(y²+4y) = −36
Complete x: 4(x²−6x+9) = 4·9=36 added outside
Complete y: 9(y²+4y+4) = 9·4=36 added outside
4(x−3)² + 9(y+2)² = −36 + 36 + 36 = 36
(x−3)²/9 + (y+2)²/4 = 1

Memory shortcuts:
• "a is for Away from center to vertex" — a is always biggest
• "Bigger denominator = direction of the stretch" — foci follow the bigger denominator
• c² = a² − b² (not +, that's hyperbola!)
• LR = 2b²/a (think: b-squared over a, times 2)
Most common mistakes:
• Using c²=a²+b² (that's a hyperbola — for ellipse it's MINUS)
• Putting a under the smaller direction (a always goes under the BIGGER stretch)
• Forgetting to flip signs when reading h and k
• Not dividing by the RHS constant to get the equation = 1 first
